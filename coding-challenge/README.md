## This is a **Gradle** project for **Spring Boot** based web service ##

## **Build** ##
`gradlew build`

## **Run** ##
`gradlew bootRun`

## **Generate Eclipse Project** ##
`gradlew eclipse`

## **Generate Idea Project** ##
`gradlew idea`
